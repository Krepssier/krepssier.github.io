Prioridad de operadores

Al igual que ocurre con las matemáticas, existe un orden de prioridad cuando se opera con los operadores. Así que para poder encontrar más fácilmente esta prioridad, dejo la tabla expuesta por si en algún momento se necesita consultar esta prioridad.

Cuando empecemos a operar con ellos, veremos de forma más detenida lo que significa la asociatividad.

Operador Asociatividad
() [] . izquierda a derecha
- ~ ! ++ -- derecha a izquierda
new (tipo) expresión derecha a izquierda
* / % izquierda a derecha
+ - izquierda a derecha
<< >> >>> izquierda a derecha
< <= > >= instance of izquierda a derecha
== != izquierda a derecha
& izquierda a derecha
^ izquierda a derecha
| izquierda a derecha
&& izquierda a derecha
|| izquierda a derecha
?: derecha a izquierda
= *= /= %= += -= <<= >>= >>>= &= |= ^= derecha a izquierda
